# **App Name**: S-Tech Services Hub

## Core Features:

- Audience Segmentation: Clear landing page sections to direct users to 'Repairs' or 'Development' services, ensuring immediate segmentation.
- Service Listings: Detailed listings of electronics and mobile repair services, including screen/battery replacements, with clear pricing and service descriptions.
- AI-Powered Solutions Showcase: Showcase of AI-driven full-stack web development and AI-enhanced cross-platform app design, detailing benefits and technologies used. The showcase uses AI as a tool to choose appropriate case studies.
- About Us Timeline: Interactive timeline highlighting S-Tech Services' history since 2014, emphasizing the blend of hardware and software expertise.
- Repair Quote System: Real-time quote generation system for local repair services. Uses ai to choose appropriate replacement parts.
- Development Consultation Scheduler: Online scheduling tool for businesses to book consultations for AI-powered web and app development services.
- Portfolio Filtering: Dynamic portfolio section with filtering to differentiate between 'Repair Projects' and 'Development Projects.'

## Style Guidelines:

- Primary color: Deep, trustworthy blue (#2979FF) to convey reliability and technological expertise.
- Background color: Light blue-gray (#F0F4F8) for a clean, modern feel.
- Accent color: Vibrant orange (#FFAB40) for calls to action and highlighting key information, providing contrast and energy.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused aesthetic.
- Body font: 'Inter' (sans-serif) for clean readability.
- Use modern, minimalist icons to represent services and features, ensuring clarity and visual appeal.
- Clean, grid-based layout with clear content sections to facilitate easy navigation and readability.
